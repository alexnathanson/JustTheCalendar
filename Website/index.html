How does its work?
The Facebook Graph API provides all of the necessary information to create Just the Calendar.

Do you store any of my data?
Nope! Just the Calendar does not store any of your Facebook data at all and we don't pull more info than is needed.

Known Issues
The Mozilla Facebook Container for Firefox is not compatible with the system we're using to log in users and retrieve calendar info. If you want to use both a container system to keep Facebook from sniffing around your life and this application we recommend using Mozilla's Firefox Multi-Account Containers add-on and adding the Just the Calendar website to the Facebook tab. This will only work for the website and not the add-on pop-up.

Some users have reported not seeing any events in their calendar and getting this error message in their console. "Cannot read property 'name' of undefined" This problem should have been resolved, but if you have received this error (after 4/4/18) please email us.
